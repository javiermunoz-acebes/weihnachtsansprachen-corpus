# Weihnachtsansprachen Corpus

This repository contains the Weihnachtsansprachen Corpus, a collection of Christmas speeches by German chancellors.

## Structure

- `speeches/` — Plain text files of the speeches, organized by chancellor
- `scripts/` — Utility scripts for corpus management
- `CORPUS_METADATA.csv` — Metadata for all speeches
- `CITATION.cff` — Citation information

## Usage

Please cite this corpus according to the `CITATION.cff` file.
